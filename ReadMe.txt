توضیح پروژه:
این پروژه یک اپلیکیشن مبدل ارز ساده و واکنش‌گرا است که به کاربران اجازه می‌دهد بین دلار آمریکا و ریال ایران تبدیل انجام دهند. نرخ تبدیل توسط کاربر قابل تغییر است و نتایج به صورت لحظه‌ای نمایش داده می‌شوند.

ویژگی‌ها

تبدیل دوطرفه: دلار → ریال و ریال → دلار
